MicroWindows是一套起源於1992年、名為ToyGUI的計劃，後來重構為
1994年時所完成的樣子。2006年時，我決定將原始碼整理之後，以三
條文式BSD授權條款（3-Clause BSD License）公開。

原始碼詳細的授權說明可參考目錄中的License/，而1994年的說明檔，
包括class hierarachy及架構說明則放於Document/目錄中。所有文件
均已轉為UTF-8編碼。

進一步的說明，可參考：http://dengliu.org/microwindows
